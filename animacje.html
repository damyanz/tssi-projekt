Animacje

Przykład 1

Animacja płynnie zmieniająca tło div'a 
<div></div>
          
div {
  width: 100px;
  height: 100px;
  background-color: red;
  animation-name: przyklad1; /* Nazwa */
  animation-duration: 4s; /* Czas trwania */
}
@keyframes przyklad1 {
  from {background-color: red;} /* Początek */
  to {background-color: yellow;} /* Koniec */
}
Efekt:

@keyframes nazwa_animacji - składnia definicji animacji

Przykład 2

<div></div>
div {
  width: 100px;
  height: 100px;
  background-color: red;
  position: relative;
  animation-name: przyklad2;
  animation-duration: 5s;
  animation-timing-function: linear; /* Regularność */
  animation-delay: 2s; /* Opóźnienie */
  animation-iteration-count: infinite; /* Powtórzenia */
  animation-direction: alternate; /* Kierunek */
}
@keyframes przyklad2 {
  0%   {background-color:red; left:0px; top:0px;}
  25%  {background-color:yellow; left:200px; top:0px;}
  50%  {background-color:blue; left:200px; top:200px;}
  75%  {background-color:green; left:0px; top:200px;}
  100% {background-color:red; left:0px; top:0px;}
}
