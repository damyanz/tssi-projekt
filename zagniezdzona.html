Lista zagnieżdżona

Krótko mówiąc, jest to po prostu "lista w liście". Przy tworzeniu należy pamiętać, aby umieścić ją pomiędzy znacznikami <li> i </li> elementu listy wyższej

Lista zagnieżdżona

<ol>
  <li>Element 1</li>
  <li>Element 2
    <ul>
      <li>Element A</li>
      <li>Element B</li>
      <li>Element C</li>
    </ul>
  </li>
  <li>Element 3</li>
<ol>
Wygląda ona następująco:
  1. Element 1
  2. Element 2
    • Element A
    • Element B
    • Element C
  3. Element 3
