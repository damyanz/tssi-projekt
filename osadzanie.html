Osadzanie CSS'a

Istnieją 3 sposoby osadzania kaskadowych arkuszów stylów w dokumencie HTML:

1. Styl lokalny - wewnątrz dowolnego znacznika

Lokalny

<selektor style="cecha: wartość; cecha2: wartość2;...">...</selektor>
Styl lokalny formatuje tylko ten pojedynczy element, w którym został osadzony

selektor
style="..." - Między cudzysłowy wpisujemy kod CSS
cecha - właściwość formatowana, np. kolor tekstu, tło, ramka, marginesy
wartość - wartość atrybutu, np. dla koloru tekstu: red, dla marginesów: 5px

2. Styl wewnętrzny - wewnątrz pliku html, pomiędzy znacznikami <style> w sekcji <head>

Wewnętrzny

<head>
  	[...]
<style>
  selektor{
    cecha: wartość;
    cecha2: wartość2;
  }
  selektor2{
    cecha: wartość;
    cecha2: wartość2;
  }
  selektor3, selektor 4{
    cecha: wartość;
    cecha2: wartość2;
  }
</style>
</head>
Styl wewnętrzny formatuje każdy element tego samego typu w obrębie dokumentu, w którym został osadzony

3. Styl zewnętrzny - osobny plik z rozszerzeniem .css dołączony do dokumentu poprzez linijkę umieszczoną w sekcji <head>

Dołączanie zewnętrznego arkusza stylów

<head>
  [...]
  <link rel="Stylesheet" type="text/css" href="style.css" />
</head>
href="..." - ścieżka do arkusza

Zewnętrzny - plik style.css

selektor{
  cecha: wartość;
  cecha2: wartość2;
}
selektor2{
  cecha: wartość;
  cecha2: wartość2;
}
selektor3, selektor 4{
  cecha: wartość;
  cecha2: wartość2;
}
Styl wewnętrzny formatuje każdy element tego samego typu w obrębię wszystkich dokumentów do których został dołączony
