Kroki tworzenia menu poziomego

Krok 1.

Tworzymy punktowaną listę odnośników 
<ul>
  <li><a href="#">Opcja 1</a></li>
  <li><a href="#">Opcja 2</a></li>
  <li><a href="#">Opcja 3</a></li>
  <li><a href="#">Opcja 4</a></li>
  <li><a href="#">Opcja 5</a></li>
</ul>
Otrzymamy:
  • Opcja 1
  • Opcja 2
  • Opcja 3

Krok 2.

Usuwamy punktory 
ul{
  list-style-type:none;
}
Otrzymamy:
  • Opcja 1
  • Opcja 2
  • Opcja 3

Krok 3.

Usuwamy podkreślenia i zmieniamy kolor tekstu 
a{
  text-decoration:none;
  color:black;
}
Otrzymamy:
  • Opcja 1
  • Opcja 2
  • Opcja 3

Krok 4.

Ustawiamy <a> jako inline-block i ustawiamy jego szerokość, natomiast <li> ustawiamy jako inline 
a{
  text-decoration:none;
  color:black;
  display:inline-block;
  width:100px;
}
li{
  display:inline;
}
Otrzymamy:
  • Opcja 1
  • Opcja 2
  • Opcja 3

Krok 5.

Zajmujemy się wyglądem, tj. ustawiamy marginesy zewnętrzne i wewnętrzne, kolor tła, ramkę, środkujemy tekst i zmieniamy jego wielkość 
a{
  text-decoration:none;
  color:black;
  display:inline-block;
  width:100px;
  margin:10px;
  padding:3px 5px;
  background-color:green;
  border:2px double black;
  text-align:center;
  font-size:1.2em;
}

Krok 6.

Ustawiamy efekt najechania myszą na element menu poprzez stworzenie nowego selektora z pseudoklasą :hover 
a:hover{
background-color:darkgreen;
}
