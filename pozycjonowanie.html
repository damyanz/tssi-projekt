Pozycjonowanie


Istnieją 4 rodzaje pozycjonowania

1. position: static;

Domyślne pozycjonowanie 
div.static {
  position: static;
  border: 3px solid #73AD21;
}
Ten div jest pozycjonowany statycznie

2. position: relative;

Element pozycjonowany jest relatywnie względem swojej normalnej pozycji 
div.relative{
  position: relative;
  left:30px;
  border: 3px solid #73AD21;
}
Ten div jest pozycjonowany relatywnie

3. position: fixed;

Element jest zawsze w tej samej pozycji względem okna przeglądarki, nawet gdy użytkownik przewija stronę. 
div.fixed{
  position: fixed;
  bottom:0;
  right:0;
  width:150px;
  border: 3px solid #73AD21;
}
Jest w prawym dolnym rogu przeglądarki :)
Ten div jest przylepiony do okna przeglądarki

4. position: absolute;

Element jest pozycjonowany relatywnie względem najbliższego pozycjonowanego elementu, w którym się znajduje. 
div.relative {
  position: relative;
  width: 400px;
  height: 200px;
  border: 3px solid #73AD21;
}
div.absolute {
  position: absolute;
  top: 80px;
  right: 0;
  width: 200px;
  height: 100px;
  border: 3px solid #73AD21;
}
Ten div jest pozycjonowany relatywnie
Ten div jest pozycjonowany absolutnie
