Gradienty

Gradient - płynne przejście pomiędzy kolorami

Przykład

<div id="grad"><div>
#grad {
  height:100px;
  background: red; /* Dla przeglądarek, które nie obsługują gradientów */
  background: -webkit-linear-gradient(red, yellow); /*Dla Safari od 5.1 do 6.0*/
  background: -o-linear-gradient(red, yellow); /* Dla Opery od 11.1 do 12.0 */
  background: -moz-linear-gradient(red, yellow); /* Dla Firefoxa od 3.6 do 15 */
  background: linear-gradient(red, yellow); /* Standardowa składnia */
}
Efekt:

background: linear-gradient(kierunek, kolor1, kolor2, ...); - składnia tworzenia gradientu liniowego

Przykład 2

<div id="grad"><div>
#grad {
  height:100px;
  background: red;
  background: -webkit-repeating-radial-gradient(red, yellow 10%, green 15%);
  background: -o-repeating-radial-gradient(red, yellow 10%, green 15%);
  background: -moz-repeating-radial-gradient(red, yellow 10%, green 15%);
  background: repeating-radial-gradient(red, yellow 10%, green 15%);
}
Efekt:

background: radial-gradient(rozmiar kształtu, kolor początkowy, ..., ostatni kolor); - składnia tworzenia gradientu okrągłego powtarzającego
