Formularze

Formularze to jedne z najciekawszych elementów stron internetowych. Pozwalają one autorom stron uzyskiwać informację od czytelników, zaś tym ostatnim, na przykład, wypełniać ankiety czy zamawiać jakieś towary i usługi. Przykładowy formularz poniżej



Jego kod prezentuje się następująco:

Przykładowy formularz

<form>
  <label>Imię: <input type="text" name="imie" value="Rafał"/></label>
  <label>Nazwisko: <input type="text" name="nazwisko"/></label>
  <label>Hasło: <input type="password" name="haslo"/></label>
  <label>Telefon: <input type="tel" name="telefon"/></label>
  <label>Poczta: <input type="email" name="email"/></label>
  <label>URL: <input type="url" name="url"/></label>
  <label>Data: <input type="date" name="data"/></label>
  <label>Numer: <input type="number" min="3" max="30" step="2" name="numer"/>
  </label>
  <label>Zakres: <input type="range" min="5" max="50" step="5" name="zakres"/>
  </label>
  <label>Kolor: <input type="color" name="kolor"/></label>
  <input type="submit" value="Wyślij" tabindex="3" accesskey="w"/>
</form>
Poniżej kod CSS formatujący formularz 
form{
  position:relative;
  width:250px;
  border-radius:10px;
  font-size:1em;
  margin:20px;
}
label{
  display:block;
  position:relative;
}
  input{
  width:150px;
  position:absolute;
  right:0;
  left:110px;
}
  input[name=imie]{
  background:red;
}
  input[name=nazwisko]{
  background:yellow;
}
  input[type=submit]{
  position:relative;
  width:152px;
}
  input[disabled=disabled]{
  background:blue;
}
<form> - znacznik otwierający i zamykający formularz
<label> - połączenie etykiety z polem
<input> - pole formularza
<input type="submit"> - przycisk wysyłający formularz

  <input type="radio" name="gender" value="male" checked> Mężczyzna<br>

Przykładowe typy pola input

Typ Opis
button Przycisk, zwykle używany do aktywacji skryptów
checkbox Pole wyboru
color Pole wyboru koloru
date Pole wyboru daty (z kalendarzem)
email Pole do wpisywania adresu e-mail
image Obrazek jako przycisk wysyłający
number Pole do wpisywania liczby
password Pole do wpisywania hasła (zakropkowane)
radio Przycisk wyboru
range Zakres
reset Przycisk czyszczenia formularza
tel Pole do wpisywania numeru telefonu
text Standardowe pole tekstowe
time Pole do wpisywania czasu
url Pole do wpisywania adresu URL
