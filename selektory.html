Selektory

Selektor elementu

Odnosi się do zawartości między danymi znacznikami 
p{
  text-align: center;
  color: red;
}

Oznaczanie elementu identyfikatorem i klasą

Identyfikator jest unikalny!!! Można go przypisać tylko do jednego elementu 
<p id="akapit">Lorem ipsum...</p>
Klasę możemy przypisać do kilku elementów 
<li class="lista">Lorem</li>
<li class="lista">ipsum</li>
<li class="lista">dolor</li>
<li class="lista">sit</li>

Selektor identyfikatora

Odnosi się do elementu oznaczonego identyfikatorem. id="..."
#nazwa_identyfikatora{
  text-align: center;
  color: red;
}

Selektor klasy

Odnosi się do wszystkich elementów oznaczonych daną klasą. class="..." 
.nazwa_klasy{
  text-align: center;
  color: red;
}
Można także określić element działania selektora klasy 
p.nazwa_klasy{
  text-align: center;
  color: red;
}
