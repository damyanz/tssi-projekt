Tabele

Przykładowa tabela w HTML:

1.1 1.2 1.3
2.1 2.2 2.3
3.1 3.2 3.3

Jej kod prezentuje się następująco:

Przykład 1

<table>
  <tr>
    <th>1.1</th>
    <th>1.2</th>
    <th>1.3</th>
  </tr>
  <tr>
    <td>2.1</td>
    <td>2.2</td>
    <td>2.3</td>
  </tr>
  <tr>
    <td>3.1</td>
    <td>3.2</td>
    <td>3.3</td>
  </tr>
</table>
Aby utworzyć ramkę tabeli, wykorzystałem poniższy kod CSS: 
table, td, th{
  border: 1px solid gray;
}
<table> - znacznik otwierający i zamykający tabelę
<tr> - znacznik wiersza tabeli
<th> - opcjonalny znacznik komórki nagłówka tabeli (domyślnie wyśrodkowane i pogrubione)
<td> - znacznik komórki tabeli

Łączenie komórek

HTML umożliwia nam łączenie komórek w tabelach.
Są dwie opcje łączenia:

1. Łączenie w poziomie

Imię i nazwisko Numer telefonu
Bill Gates 55577854 55577855
Kod powyższej tabeli prezentuje się następująco

Colspan

<table id="tabela_colspan">
  <tr>
    <th>Imię i nazwisko</th>
    <th colspan="2">Numer telefonu</th>
  </tr>
  <tr>
    <td>Bill Gates</td>
    <td>55577854</td>
    <td>55577855</td>
  </tr>
</table>
Poniższy kod CSS sprawia, że tabela ma 100% szerokości głównego div'a oraz komórki nagłówki mają szare tło: 
#tabela_colspan{
  width:100%;
}
#tabela_colspan th{
  background-color: #ccc;
}
colspan="..." - argument <td> / <th> definiujący liczbę komórek do połączenia w poziomie


2. Łączenie w pionie
Godziny otwarcia Dzień
pon. - pt. 8-16
sob.
niedz. nieczynne
Kod powyższej tabeli prezentuje się następująco

Rowspan

<table id="tabela_rowspan">
  <tr>
    <th>Godziny otwarcia</th>
    <th>Dzień</th>
  </tr>
  <tr>
    <td>pon. - pt.</td>
    <td rowspan="2" style="text-align:center;">8-16</td>
  </tr>
  <tr>
    <td>sob.</td>
  </tr>
  <tr>
    <td>niedz.</td>
    <td style="text-align:center;">nieczynne</td>
  </tr>
</table>;
Poniższy kod CSS sprawia, że tabela ma 100% szerokości głównego div'a, pierwszy wiersz ma tło koloru szarego, komórki mają wewnętrzne odstępy, ramki są zielone, kropkowane i rogi tabeli są zaokrąglone: 
#tabela_rowspan{
  width:100%;
  border-style:dotted;
  border-color:green;
  border-radius: 5px;
}
#tabela_rowspan td{
  border-style:dotted;
  padding:5px 10px;
  border-color:green;
}
#tabela_rowspan th{
  background-color: #ccc;
  border-style:dotted;
  border-color:green;
}
rowspan="..." - argument <td> / <th> definiujący liczbę komórek do połączenia w pionie
