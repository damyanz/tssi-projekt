Przekształcenia

Przykład 1

<div></div>
div {
  width: 100px;
  height: 100px;
  background: red;
  -webkit-transition: width 2s, height 3s;
  transition: width 2s, height 3s;
}
div:hover {
  width: 300px;
  height: 50px;
}
Efekt:

transition: właściwość czas, właściwość2 czas2; - składnia przekształcenia

Przykład 2

<div></div>
div {
  width: 100px;
  height: 100px;
  background: red;
  transition-property: width; /* Właściwość ulegająca przekształceniu */
  transition-duration: 2s; /* Czas trwania */
  transition-timing-function: linear; /* Regularność */
  transition-delay: 1s; /* Opóźnienie */
}
div:hover {
  width: 300px;
}
Efekt:

Przykład 3

<div></div>
<div id="obrocony"></div>
div {
  width: 300px;
  height: 100px;
  background-color: yellow;
  border: 1px solid black;
}
div#obrocony {
  transform: rotate(20deg);
}
Efekt:
Normalny div
Div obrócony



transform: akcja(wartość); - składnia przekształcenia
