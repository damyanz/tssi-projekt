Oznaczenia

Przykładowe fragmenty kodu HTML w niniejszym kursie zostały oznaczone ramką z pomarańczową linią po lewej stronie:

Kod HTML

<!DOCTYPE html>
<html>
  <head>
  	<meta charset="utf-8" />
  	<title>Tu wpisz tytuł strony</title>
  </head>
  <body>

  <p>Tu wpisuje się treść strony</p>

  </body>
  </html>
natomiast fragmenty kodu CSS zostały oznaczone ramką z granatową linią po lewej stronie:

Kod CSS

#pojemnik{
    width:500px;
    margin:0 auto;
    background-color:gray;
}

Ważne znaczniki, atrybuty i właściwości zostały oznaczone kolorem czerwonym i szarym tłem:
name="..."
<li>
width="..."
